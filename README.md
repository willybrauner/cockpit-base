# Cockpit base

# About

[Cockpit CMS](https://getcockpit.com/) base project.  

# Installation

Clone repos:
````shell script
$ git clone git@github.com:willybrauner/cockpit-base.git
````

Move to repos folder:
```shell script
$ cd cockpit-base
```

Start install script:
```shell script
$ ./install.sh
```
